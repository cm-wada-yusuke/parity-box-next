# 抽出

## [HARIO (ハリオ) 浸漬式 ドリッパー スイッチ](https://amzn.to/3tMBVkD)

| 項目 | 値 | 
|:---:|:---:|
| みるっ子メモリ | 5メモリ |
| 豆量 | 15g |
| 湯温度 | 85度 |
| 湯量 | 250cc |
| 投入回数 | 1（浸潤式マニュアル） |
| 一投あたり待機時間 | 3分 |


中南米のコーヒーらしく、マイルドですべての特徴（酸味、甘み、苦味、香り）が平均的。ドリップも普通に飲めるが、エスプレッソやアメリカーノで油分も一緒に出してしまったほうがとろみも加わって印象深くなるかもしれない。
